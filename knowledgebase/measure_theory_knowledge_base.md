## Part I: Foundational Definitions

### 1. σ-Algebra (MeasurableSpace)

**Natural Language Statement:**
A σ-algebra on a set α is a collection of subsets that: (1) contains the empty set, (2) is closed under complementation, and (3) is closed under countable unions.

**Lean 4 Definition:**
```lean
class MeasurableSpace (α : Type*) : Type* where
  MeasurableSet' : Set α → Prop
  measurableSet_empty : MeasurableSet' ∅
  measurableSet_compl : ∀ s, MeasurableSet' s → MeasurableSet' sᶜ
  measurableSet_iUnion : ∀ f : ℕ → Set α, (∀ i, MeasurableSet' (f i)) →
    MeasurableSet' (⋃ i, f i)
```

**Mathlib Support:** FULL
- **Import:** `Mathlib.MeasureTheory.MeasurableSpace.Defs`

**Key Theorems:**

### 2. Generated σ-Algebra

**Natural Language Statement:**
The σ-algebra generated by a collection of sets S is the smallest σ-algebra containing all sets in S. It can be constructed inductively by adding complements and countable unions.

**Lean 4 Definition:**
```lean
def MeasurableSpace.generateFrom (s : Set (Set α)) : MeasurableSpace α where
  MeasurableSet' := GenerateMeasurable s
  measurableSet_empty := GenerateMeasurable.empty
  measurableSet_compl := GenerateMeasurable.compl
  measurableSet_iUnion := GenerateMeasurable.iUnion

-- Inductive construction
inductive GenerateMeasurable (s : Set (Set α)) : Set α → Prop
  | basic : ∀ u ∈ s, GenerateMeasurable s u
  | empty : GenerateMeasurable s ∅
  | compl : ∀ s', GenerateMeasurable s s' → GenerateMeasurable s s'ᶜ
  | iUnion : ∀ f : ℕ → Set α, (∀ n, GenerateMeasurable s (f n)) →
      GenerateMeasurable s (⋃ n, f n)
```

**Mathlib Support:** FULL
- **Import:** `Mathlib.MeasureTheory.MeasurableSpace.Defs`

**Difficulty:** medium

### 3. Measurable Sets

### 4. Measurable Functions

### 6. Measure

**Natural Language Statement:**
A measure μ on a measurable space assigns a non-negative extended real number to each measurable set, such that: (1) μ(∅) = 0, and (2) μ is countably additive on disjoint sets.

**Lean 4 Definition:**
```lean

  extends OuterMeasure α where
  m_iUnion ⦃f : ℕ → Set α⦄ (hf : ∀ i, MeasurableSet (f i))
    (hd : Pairwise (Disjoint on f)) :
    toOuterMeasure (⋃ i, f i) = ∑' i, toOuterMeasure (f i)
  trimmed : toOuterMeasure.trim = toOuterMeasure
```

**Mathlib Support:** FULL
- **Import:** `Mathlib.MeasureTheory.Measure.MeasureSpaceDef`

**Key Properties:**

### 7. Lebesgue Measure

**Natural Language Statement:**
Lebesgue measure on ℝ assigns to each interval its length. On ℝⁿ, it equals the product of coordinate Lebesgue measures.

**Lean 4 Theorems:**
```lean
-- Intervals have measure equal to length
theorem Real.volume_Ico (a b : ℝ) : volume (Set.Ico a b) = ENNReal.ofReal (b - a)
theorem Real.volume_Icc (a b : ℝ) : volume (Set.Icc a b) = ENNReal.ofReal (b - a)
theorem Real.volume_Ioo (a b : ℝ) : volume (Set.Ioo a b) = ENNReal.ofReal (b - a)

-- Singletons have measure zero
theorem Real.volume_singleton {a : ℝ} : volume ({a} : Set ℝ) = 0

-- Product boxes in ℝⁿ
theorem Real.volume_Icc_pi [Fintype ι] (a b : ι → ℝ) :
  volume (Set.Icc a b) = ∏ i, ENNReal.ofReal (b i - a i)

-- Linear maps scale by |det|
theorem Real.map_linearMap_volume_pi_eq_smul_volume_pi
  [FiniteDimensional ℝ E] (f : E →ₗ[ℝ] E) :
  Measure.map f volume = ENNReal.ofReal |LinearMap.det f| • volume
```

- **Import:** `Mathlib.MeasureTheory.Measure.Lebesgue.Basic`

### 10. Lebesgue Integral (lintegral)

**Natural Language Statement:**
The Lebesgue integral of a non-negative measurable function f is defined as the supremum of integrals of simple functions bounded above by f.

**Lean 4 Definition:**
```lean
noncomputable def lintegral {α : Type*} {m : MeasurableSpace α}
  (μ : Measure α) (f : α → ENNReal) : ENNReal :=
  ⨆ (g : SimpleFunc α ENNReal), ⨆ (_ : ⇑g ≤ f), g.lintegral μ
```

**Notation:**
- `∫⁻ x, f x ∂μ` — integral with respect to μ
- `∫⁻ x in s, f x ∂μ` — integral over set s

**Mathlib Support:** FULL
- **Import:** `Mathlib.MeasureTheory.Integral.Lebesgue.Basic`

**Key Properties:**
```lean
-- Monotonicity
theorem lintegral_mono {f g : α → ENNReal} (hfg : f ≤ g) :
  ∫⁻ a, f a ∂μ ≤ ∫⁻ a, g a ∂μ

-- Constants
```

### 11. Bochner Integral

**Natural Language Statement:**
The Bochner integral extends Lebesgue integration to functions taking values in a Banach space. A function is integrable if it is strongly measurable and has finite integral of its norm.

**Lean 4 Definition:**
```lean
-- Integrability
def Integrable {α : Type*} {E : Type*}
  [MeasurableSpace α] [NormedAddCommGroup E]
  (f : α → E) (μ : Measure α := by volume_tac) : Prop :=
  AEStronglyMeasurable f μ ∧ ∫⁻ a, ‖f a‖₊ ∂μ < ∞

-- The integral
noncomputable def integral {α : Type*} {E : Type*}
  [MeasurableSpace α] [NormedAddCommGroup E] [NormedSpace ℝ E]
  (μ : Measure α) (f : α → E) : E
```

**Notation:**
- `∫ a, f a ∂μ` — Bochner integral
- `∫ a in s, f a ∂μ` — integral over set s

**Mathlib Support:** FULL
- **Import:** `Mathlib.MeasureTheory.Integral.Bochner.Basic`

**Key Properties:**
```lean
-- Linearity
theorem integral_add {f g : α → E}
  (hf : Integrable f μ) (hg : Integrable g μ) :
  ∫ a, f a + g a ∂μ = ∫ a, f a ∂μ + ∫ a, g a ∂μ

theorem integral_smul (c : ℝ) (f : α → E) :
  ∫ a, c • f a ∂μ = c • ∫ a, f a ∂μ
```

### 12. Almost Everywhere

### 14. Fatou's Lemma

## Part IV: Lᵖ Spaces

### 16. Lᵖ Space Definition

**Natural Language Statement:**
For 1 ≤ p < ∞, Lᵖ(μ) consists of measurable functions f with ∫|f|ᵖ < ∞. For p = ∞, it consists of essentially bounded functions. Lᵖ is a Banach space.

**Lean 4 Definition:**
```lean
-- Membership predicate
def Memℓp (f : α → E) (p : ℝ≥0∞) (μ : Measure α) : Prop :=
  AEStronglyMeasurable f μ ∧ snorm f p μ < ∞

-- Lp space (quotient by a.e. equality)
def Lp (E : Type*) [NormedAddCommGroup E]
  (p : ℝ≥0∞) (μ : Measure α) : Type* :=
  { f : α →ₘ[μ] E // Memℓp f p μ }

-- The Lp norm (snorm)
noncomputable def snorm (f : α → E) (p : ℝ≥0∞) (μ : Measure α) : ℝ≥0∞ :=
  if p = 0 then 0
  else if p = ∞ then essSup (fun x => ‖f x‖₊) μ
  else (∫⁻ a, ‖f a‖₊ ^ p.toReal ∂μ) ^ (1 / p.toReal)
```

**Mathlib Support:** FULL
- **Import:** `Mathlib.MeasureTheory.Function.LpSpace`

**Key Properties:**
```lean
-- Lp is a Banach space
instance [Fact (1 ≤ p)] : NormedAddCommGroup (Lp E p μ)
instance [Fact (1 ≤ p)] : NormedSpace ℝ (Lp E p μ)
instance [Fact (1 ≤ p)] : CompleteSpace (Lp E p μ)

-- L² is a Hilbert space
instance : InnerProductSpace ℝ (Lp ℝ 2 μ)
```

**Difficulty:** hard

### 17. Hölder's Inequality

**Natural Language Statement:**
For conjugate exponents p and q (1/p + 1/q = 1), ∫|fg| ≤ ‖f‖_p · ‖g‖_q.

**Lean 4 Theorem:**
```lean
theorem ENNReal.lintegral_mul_le_Lp_mul_Lq (μ : Measure α)
  {p q : ℝ} (hpq : p.HolderConjugate q)
  {f g : α → ENNReal} (hf : AEMeasurable f μ) (hg : AEMeasurable g μ) :
  ∫⁻ a, (f * g) a ∂μ ≤
    (∫⁻ a, f a ^ p ∂μ) ^ (1/p) * (∫⁻ a, g a ^ q ∂μ) ^ (1/q)

-- For snorm
theorem snorm_mul_le {f : α → E} {g : α → F}
  (hf : AEStronglyMeasurable f μ) (hg : AEStronglyMeasurable g μ)
  {p q r : ℝ≥0∞} (hp : p ≠ 0) (hq : q ≠ 0)
  (h : 1/p + 1/q = 1/r) :
  snorm (fun x => ‖f x‖ * ‖g x‖) r μ ≤ snorm f p μ * snorm g q μ
```

### 19. Fubini's Theorem

**Natural Language Statement:**

### 20. Radon-Nikodym Theorem

**Natural Language Statement:**
If μ ≪ ν (μ is absolutely continuous w.r.t. ν), then there exists a measurable function f (the Radon-Nikodym derivative) such that dμ = f dν.

**Lean 4 Theorems:**
```lean
-- Absolute continuity
def AbsolutelyContinuous (μ ν : Measure α) : Prop :=
  ∀ s, MeasurableSet s → ν s = 0 → μ s = 0

notation:50 μ " ≪ " ν => AbsolutelyContinuous μ ν

-- Radon-Nikodym derivative
noncomputable def rnDeriv (μ ν : Measure α) : α → ℝ≥0∞

-- Main theorem
theorem absolutelyContinuous_iff_withDensity_rnDeriv_eq
  {μ ν : Measure α} [HaveLebesgueDecomposition μ ν] :
  μ ≪ ν ↔ ν.withDensity (μ.rnDeriv ν) = μ

-- Integration with RN derivative
theorem setLIntegral_rnDeriv {μ ν : Measure α}
  [HaveLebesgueDecomposition μ ν] (hμν : μ ≪ ν)
  {s : Set α} (hs : MeasurableSet s) :
  ∫⁻ x in s, μ.rnDeriv ν x ∂ν = μ s
```

### 21. Lebesgue Decomposition

**Natural Language Statement:**
Any measure μ can be uniquely decomposed as μ = μₛ + μₐ where μₛ ⊥ ν (singular) and μₐ ≪ ν (absolutely continuous).

**Lean 4 Definition:**
```lean
class HaveLebesgueDecomposition (μ ν : Measure α) : Prop where
  lebesgue_decomposition :
    ∃ p : Measure α × Measure α,
      MutuallySingular p.1 ν ∧ p.2 ≪ ν ∧ μ = p.1 + p.2
```

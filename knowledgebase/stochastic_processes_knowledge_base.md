### Section 1.1: Filtration Definition and Basic Properties (5-6 statements)

#### 1. Filtration

**Natural Language Statement:**
A filtration on a measurable space indexed by a preordered set is a monotone sequence of sub-σ-algebras, representing the evolution of information over time.

**Lean 4 Definition:**
```lean

  [Preorder ι] (m : MeasurableSpace Ω) : Type (max u₁ u₂) where
  seq : ι → MeasurableSpace Ω
  mono' : Monotone seq
  le' (i : ι) : seq i ≤ m
```

**Mathlib Location:** `Mathlib.Probability.Process.Filtration`

**Key Theorems:**
- `Filtration.mono : i ≤ j → f i ≤ f j` (monotonicity)
- `Filtration.le : f i ≤ m` (sub-σ-algebra property)

#### 18. Stopping Time

**Natural Language Statement:**

### Section 5.1: Martingale Definitions (4-5 statements)

#### 37. Martingale

**Natural Language Statement:**
A stochastic process is a martingale with respect to a filtration if it is adapted and the conditional expectation at any earlier time equals the present value, representing a fair game where future expected values equal current values.

```lean
  Adapted ℱ f ∧ ∀ (i j : ι), i ≤ j → μ[f j | ↑ℱ i] =ᵐ[μ] f i
```

#### 65. Tower Property of Conditional Expectation

**Natural Language Statement:**

### Section 9.1: Almost Everywhere Convergence (1-2 statements)

#### 69. Doob's Almost Everywhere Martingale Convergence Theorem

**Natural Language Statement:**
An L¹-bounded submartingale adapted to a filtration converges almost everywhere to a limit process, one of the fundamental convergence results in martingale theory showing that bounded submartingales have well-defined limits.

**Lean 4 Theorem (from literature):**
```lean
theorem MeasureTheory.Submartingale.ae_tendsto_limitProcess
  {Ω : Type u_1} {m : MeasurableSpace Ω} {μ : Measure Ω}
  {ℱ : Filtration ℕ m} {f : ℕ → Ω → ℝ}
  (hf : Submartingale f ℱ μ)
  (hbdd : ∃ C, ∀ n, ∫ ω, ‖f n ω‖ ∂μ ≤ C) :
  ∀ᵐ ω ∂μ, ∃ L, Filter.Tendsto (fun n => f n ω) Filter.atTop (nhds L)
```

**Mathlib Location:** `Mathlib.Probability.Martingale.Convergence`

**Difficulty:** hard

**Note:** This is the classical Doob martingale convergence theorem, formalized in Mathlib.

### Section 9.2: Lp Convergence (1 statement)

#### 70. Limit Process Lp Property

**Natural Language Statement:**
The limit process of an Lᵖ-bounded submartingale is itself in Lᵖ, showing that convergence preserves integrability properties.

**Lean 4 Theorem (from literature):**
```lean
theorem MeasureTheory.Submartingale.mem_ℒp_limitProcess
```
